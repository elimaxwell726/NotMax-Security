# Simple Python Port Scanner

A lightweight TCP port scanner written in Python.  
This tool performs a sequential scan of common ports (1–1024) and reports which ones are open on a target host. It’s designed as a foundational security script for learning networking, socket programming, and basic reconnaissance techniques.

---

## Features

- Scans TCP ports 1–1024  
- Uses Python’s built‑in `socket` library  
- Detects open ports with simple connect attempts  
- Graceful error handling  
- Beginner‑friendly code structure  
- Easy to extend with threading, banner grabbing, or service detection  

---

## Requirements

- Python 3.x  
- No external libraries required  

---
